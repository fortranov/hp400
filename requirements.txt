# Зависимости для HP Scanner Counter Script

# Основные зависимости (встроенные модули):
# - socket (для сетевого подключения)
# - argparse (для CLI интерфейса)
# - sys (для системных операций)
# - time (для задержек)
# - typing (для типизации)
# - subprocess (для системных команд)
# - platform (для определения ОС)
# - tempfile (для временных файлов)
# - os (для файловых операций)

# Дополнительные зависимости для USB подключения (опционально):
pyusb>=1.0.0

# Рекомендуется Python 3.6+

# Установка зависимостей:
# pip install pyusb

# На Linux дополнительно может потребоваться:
# sudo apt-get install libusb-1.0-0-dev
# или
# sudo yum install libusb1-devel

# На Windows может потребоваться установка драйверов libusb
